Setup
=====

Copy env.template to env and edit

Edit run_ipt_s3.sh

Run run_ipt_s3.sh

NB! This only copies your local IPT data to the S3 host. (One way data flow.)
Edit run_ipt_s3.sh to enable initial copying of data from S3 host.
May not work as a scipt and may need to be run BEFORE tomcat starts!!!
